# 🌙 Dark Mode Toggle System

A simple and responsive **Dark Mode Toggle System** built using **HTML, CSS, and JavaScript**. This project allows users to switch between **Light Mode** and **Dark Mode** dynamically with a smooth and user-friendly interface.

## 🚀 Features

- 🌞 Light Mode and 🌙 Dark Mode toggle
- 🎨 Clean and responsive UI
- ⚡ Smooth theme switching
- 💾 Theme preference persistence (if local storage is implemented)
- 📱 Mobile-friendly design

## 🛠️ Technologies Used

- **HTML5** – Structure of the webpage
- **CSS3** – Styling and theme design
- **JavaScript** – Toggle functionality and interactivity

## 📂 Project Structure

```plaintext
Dark-Mode-Toggle-System-main
│── index.html
│── style.css
│── script.js
│── assets/ (if available)
└── README.md
```

## ⚙️ How It Works

1. The webpage loads in the default theme.
2. Users can click the **toggle button/switch**.
3. JavaScript changes the theme dynamically.
4. CSS updates colors and styles for dark/light mode.
